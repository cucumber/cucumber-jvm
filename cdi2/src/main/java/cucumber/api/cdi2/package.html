There is no API for this module, but by including the cucumber-cdi2 jar on your CLASSPATH your Step Definitions will be instantiated by CDI SE API.
For this module to work you must provide:
  • A CDI-API dependency (version >= 2)
  • A CDI implementation dependency (
    org.apache.openwebbeans:openwebbeans-se:2.0.10
    for example)
